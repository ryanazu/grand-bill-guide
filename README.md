# Project Name

A modern web application built with React, TypeScript, and Tailwind CSS.

## Getting Started

Make sure you have Node.js and npm installed. If not, you can [install them via nvm](https://github.com/nvm-sh/nvm#installing-and-updating).

```sh
# Clone the repo
git clone <YOUR_GIT_URL>

# Navigate into the project
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm i

# Start the dev server
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Contributing

A few ways to make changes:

**Locally** — Clone the repo, make your changes in your editor of choice, and push.

**In GitHub** — Open a file, hit the pencil icon to edit, and commit directly.

**With Codespaces** — Spin up a Codespace from the repo's main page and work from the browser.

## Deployment

You can deploy this project to any static hosting provider (Vercel, Netlify, Cloudflare Pages, etc.). Just point it at this repo and set the build command to `npm run build` with the output directory set to `dist`.
